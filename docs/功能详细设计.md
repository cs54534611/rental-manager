# 功能详细设计文档

## 一、微信通知推送

### 1.1 功能需求
- 租金到期提醒（提前3天/当天）
- 合同到期提醒（提前30天/7天）
- 报修状态变更通知
- 缴费成功通知

### 1.2 技术方案
```javascript
// 方案一：微信订阅消息（推荐）
// 优点：无需服务器，主动推送
// 缺点：需要用户主动订阅

// 方案二：微信服务号模板消息
// 优点：可批量推送
// 缺点：需要认证服务号

// 方案三：uni-push 统一推送
// 优点：支持多平台
// 缺点：需要配置
```

### 1.3 数据库设计
```sql
-- 通知模板配置表
CREATE TABLE `notify_templates` (
  `id` INT PRIMARY KEY AUTO_INCREMENT,
  `type` VARCHAR(50) NOT NULL, -- 'rent_due', 'contract_expiring', 'repair_status'
  `template_id` VARCHAR(100), -- 微信模板ID
  `title` VARCHAR(100),
  `content` VARCHAR(500),
  `enabled` TINYINT DEFAULT 1,
  `created_at` DATETIME
);

-- 用户通知记录表
CREATE TABLE `notify_logs` (
  `id` INT PRIMARY KEY AUTO_INCREMENT,
  `user_id` INT,
  `type` VARCHAR(50),
  `title` VARCHAR(100),
  `content` VARCHAR(500),
  `status` TINYINT DEFAULT 0, -- 0待发送 1已发送 2失败
  `send_time` DATETIME,
  `created_at` DATETIME
);
```

### 1.4 API 设计
| 路由 | 方法 | 说明 |
|------|------|------|
| `/api/notify/subscribe` | POST | 用户订阅 |
| `/api/notify/send` | POST | 发送通知 |
| `/api/notify/logs` | GET | 通知记录 |

```javascript
// 推荐：阿里云 OSS 或 腾讯云 COS
```

```env
COS_SECRET_ID=your_id
COS_SECRET_KEY=your_key
COS_BUCKET=rental-images
COS_REGION=ap-chengdu
```

### 2.4 代码实现思路
```javascript
// 上传组件封装
const uploadFile = async (filePath, type = 'houses') => {
  const ext = filePath.split('.').pop();
  const key = `${type}/${Date.now()}.${ext}`;
  // 调用OSS上传
  const url = await oss.upload(key, filePath);
  return url;
};
```

## 三、多管理员权限

### 3.1 功能需求
- 超级管理员（创建者）：全部权限
- 普通管理员：查看、操作用户数据
- 财务人员：仅租金相关
- 维修人员：仅报修相关

### 3.2 角色权限表
| 角色 | 房源 | 租客 | 合同 | 租金 | 报修 | 系统设置 |
|------|------|------|------|------|------|----------|
| 超级管理员 | 全部 | 全部 | 全部 | 全部 | 全部 | 全部 |
| 普通管理员 | 全部 | 全部 | 全部 | 全部 | 全部 | 查看 |
| 财务人员 | 查看 | 查看 | 查看 | 全部 | - | - |
| 维修人员 | 查看 | - | - | - | 全部 | - |

### 3.3 数据库设计
```sql
CREATE TABLE `admin_users` (
  `id` INT PRIMARY KEY AUTO_INCREMENT,
  `username` VARCHAR(50) UNIQUE,
  `password` VARCHAR(100),
  `role` VARCHAR(20), -- super, admin, finance, repair
  `name` VARCHAR(50),
  `phone` VARCHAR(20),
  `status` TINYINT DEFAULT 1,
  `created_at` DATETIME
);

-- 权限配置表
CREATE TABLE `permissions` (
  `id` INT PRIMARY KEY AUTO_INCREMENT,
  `role` VARCHAR(20),
  `module` VARCHAR(20), -- houses, tenants, contracts, rentals, repairs
  `actions` VARCHAR(100) -- create,read,update,delete
);
```

### 3.4 登录流程
```javascript
// 登录接口
POST /api/auth/login
{ username, password }

// 返回
{
  token: 'xxx',
  user: { id, username, role, name }
}

// 中间件验证
const auth = (roles = []) => {
  return (req, res, next) => {
    const token = req.headers.authorization;
    const user = jwt.verify(token);
    if (!roles.includes(user.role)) {
      return res.status(403).json({ message: '无权限' });
    }
    next();
  };
};
```

## 四、收款功能

### 4.1 功能需求
- 生成收款二维码
- 记录收款信息
- 租金自动匹配

### 4.2 微信支付配置
```env
WECHAT_MCH_ID=商户号
WECHAT_API_KEY=API密钥
WECHAT_NOTIFY_URL=回调地址
```

### 4.3 核心流程
```
1. 生成收款二维码（租金ID + 金额）
2. 用户扫码支付
3. 微信回调通知
4. 系统更新租金状态为"已缴"
```


```sql
CREATE TABLE `checkouts` (
  `id` INT PRIMARY KEY AUTO_INCREMENT,
  `contract_id` INT,
  `apply_date` DATETIME,
```
